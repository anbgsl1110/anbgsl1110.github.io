1 linux下安装texlive2016

1.1 依赖包

  • 安装过程中需要调用Perl的模块Digest;MD5来检测ISO文件的完整性
  • 升级过程需要调用Perl的模块TK

1.1.1 CentOS下安装依赖包

yum install perl-Digest-MD5 yum install perl-TK

1.1.2 Ubuntu下安装依赖包

apt-get install libdigest-perl-md5-perl apt-get install perl-tk

1.2 安装

1.2.1 下载

  • 下载地址：
    • 官方镜像：http://mirrors.ctan.org/systems/texlive/Images/texlive2016.iso
    • USTC镜像: http://mirrors.ustc.edu.cn/CTAN/systems/texlive/Images/texlive2016.iso

1.2.2 挂载ISO镜像

su mount -o loop texlive2016.iso /mnt/ cd /mnt ./install-t1

出现选项后，输入I直接安装(也可以更改选项)。

1.2.3 环境变量

在当前用户的 

~/.bashrc

中加入如下语句 

# Tex live 2016 export MANPATH=${MANPATH}:/urs/local/texlive/2016/texmf-dist/doc/man export INFOPATH=${INFOPATH}:/urs/local/texlive/2016/texmf-dist/doc/info
              export PATH=${PATH}:/urs/local/texlive/2016/bin/x86_64-linux

1.2.4 卸载ISO镜像

cd umount /mnt/

1.3 更新

1.3.1 更新TexLive包管理器

tlmgr update --self

1.3.2 更新TexLive全部包

tlmgr update --all
