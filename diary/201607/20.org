* github上创建分支
新建好的代码库有且仅有一个主分支（master），它是自动建立的。

可以新建分支用于开发：
#+BEGIN_SRC git
git branch develop master
#+END_SRC
新建一个叫develop的分支，基于master分支

切换到这个分支：
#+BEGIN_SRC git
git checkout develop
#+END_SRC
现在可以在这个develop分支上做一些改动，并且提交。

注意：切换分支的时候可以发现，在Windows中的repository文件夹中的文件内容也会实时相应改变，变成当前分支的内容。
